# Expense AI Tracker

A modern mobile expense tracking application built with Flutter.
This app helps users manage their finances, track expenses, and automatically detect transactions from receipts using AI.

---

## 🚀 Features (Planned)

- 📊 Expense dashboard
- 💰 Income & expense tracking
- 📷 Receipt scanning using camera
- 🤖 AI-powered receipt parsing
- 📈 Expense analytics
- ☁️ Cloud sync
- 🔐 Authentication

---

## 🛠 Tech Stack

- **Frontend:** Flutter
- **State Management:** Provider (initially)
- **Backend API:** Python / FastAPI (planned)
- **AI Processing:** Gemini API (planned)
- **Database:** PostgreSQL / Firebase (TBD)

## 📦 Installation

1. Clone repository

```
git clone <repo-url>
```

2. Go to project folder

```
cd mobile/flutter_app
```

3. Install dependencies

```
flutter pub get
```

4. Run the app

```
flutter run
```
